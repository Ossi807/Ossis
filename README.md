# LaneForge

LaneForge is an original, offline Android lane-battler prototype inspired by the general real-time card-arena genre.

## Included
- Portrait touch controls
- Two lanes and two bridges
- Two guard towers plus a core tower per side
- Eight playable cards
- Energy regeneration and double-energy final minute
- Offline bot opponent
- Win/loss statistics stored locally
- No network permission and no account required

## Build

```bash
gradle :app:assembleDebug
```

The debug APK is written to `app/build/outputs/apk/debug/app-debug.apk`.
